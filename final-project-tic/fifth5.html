5)Html et Css

HTML

Le HTML (Langage de Balisage HyperTexte) est le langage fondamental qui permet de créer la structure et le contenu d'une page web. Il utilise des balises pour définir les éléments de la page : titres, paragraphes, images, liens, etc. Contrairement à un langage de programmation, le HTML se contente d'organiser l'information. Il est ensuite mis en forme par le CSS et rendu interactif par le JavaScript. En bref : le HTML est le squelette indispensable de tout site internet. Sans lui, aucune page web ne peut exister.


Css

Le CSS (Cascading Style Sheets) est le langage qui habille et met en forme les pages web. Alors que le HTML crée la structure (le squelette), le CSS définit l'apparence : couleurs, polices, espacements, mises en page et effets visuels. Son principe est simple : il cible des éléments HTML (via des sélecteurs) pour leur appliquer des propriétés de style. Une même feuille de style peut contrôler l'apparence de tout un site, garantissant une cohérence visuelle parfaite. Le CSS est l'artiste graphique du web. Il transforme une structure HTML brute en une interface élégante, agréable et lisible pour l'utilisateur
